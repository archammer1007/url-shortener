Retrieving a short url

Enter a new website by appending /new/[website address] to this page's url.

Valid website addresses should be in the format http(s)://[website]

For example, to retrieve a short url for http://www.google.ca enter https://stormy-plains-31721.herokuapp.com/new/http://www.google.ca

The service will respond with a short url for the website entered

Using a short url

To visit a website using the short url, append /[the short url] to this page's url

For example, if the short url for http://www.google.ca is 7, enter https://stormy-plains-31721.herokuapp.com/url-shortener-microservice/7
